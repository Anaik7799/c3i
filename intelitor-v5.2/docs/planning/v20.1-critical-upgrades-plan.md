- No circular buffer for pre-event capture

### Implementation

**New Files:**
```
lib/indrajaal/video/preroll/

  buffer_manager.ex     # GenServer managing per-camera buffers
  event_trigger.ex      # Alarm-triggered buffer freeze
```

**STAMP Constraints:**
- SC-PREROLL-001: Non-blocking write < 100us
- SC-PREROLL-002: 30-60 second lookback configurable
- SC-PREROLL-003: One buffer per active stream
