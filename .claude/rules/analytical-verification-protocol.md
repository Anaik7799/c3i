## The 4-Phase Protocol (13 Controls)

### Phase 01 — Provenance & Fact-Checking (Journalism + Law)
1. **Two-Source Rule** — ≥2 independent sources per major claim
2. **Hearsay Check** — Primary data vs secondary aggregation
3. **Admiralty Validation** — Rate sources A1-F6, reject below B3
4. **Chain of Custody** — Raw Data → Transformation → Conclusion (no black-box leaps)

### Phase 02 — The Interrogation (Legal + Intelligence)
5. **Cross-Examination** — 3-5 varied questions, document inconsistencies
6. **Tenth Man Activation** — Build strongest counter-case, rate Weak/Moderate/Strong
7. **MMO Assessment** — Means + Motive + Opportunity (all three required)

### Phase 03 — Structural Logic (Math + Philosophy)
8. **Proof by Contradiction** — Assume advice is terrible, trace consequences
9. **Falsifiability Test** — What evidence would prove it wrong? If none → reject
10. **ACH (Competing Hypotheses)** — Multiple explanations weighed, not just preferred

### Phase 04 — Output Evaluation (Info Theory + Statistics)
11. **Shannon Entropy** — Net-new insight rating (High/Med/Low/Zero)
12. **Type I vs Type II Error** — Which error is more costly in THIS context?
13. **Probability Yardstick** — Convert verbal hedges to explicit percentages
