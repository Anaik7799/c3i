1. Quick Start

Build Pi

cd sub-projects/pi-mono
npm install
npm run build

Run Pi CLI

node packages/coding-agent/dist/cli.js --help
# Or:
./pi-test.sh

With Providers

# Google (default)
pi --provider google --model gemini-2.5-flash

# Anthropic
pi --provider anthropic --model claude-sonnet-4-6

# Local Ollama
pi --provider openai --model gemma3 \
  --api-key ollama \
  --base-url http://localhost:11434/v1

Set API keys via environment variables:
ANTHROPIC_API_KEY, OPENAI_API_KEY, GOOGLE_API_KEY

C3I Integration via Extensions

The C3I bridge runs as a Pi extension that:

  • Publishes all events to Zenoh mesh
  • Registers 73 MCP tools as Pi tools
  • Gates L0 tools via Guardian
  • Syncs sessions to Smriti.db
  • Injects OTel spans for tracing

Install Extensions

pi install github:user/extension
pi list
pi remove github:user/extension

Troubleshooting

tsgo build fails → npm install @typescript/native-preview-linux-x64

No API key → Set ANTHROPIC_API_KEY etc.

TUI garbled → TERM=xterm-256color

Sessions not persisting → Check Smriti bridge

Zenoh events missing → Check router on TCP 7447

Tools not found → Verify c3i-bridge extension loaded
