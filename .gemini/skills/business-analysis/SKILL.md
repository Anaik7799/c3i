# Business Analysis — Analyze a market, company, or business problem

Perform structured business analysis on "$ARGUMENTS":

## 1. Situation Assessment
- What is the core question or problem?
- Who are the stakeholders and what do they care about?
- What data is available vs what needs to be gathered?
- What decisions will this analysis inform?

## 2. Market Analysis (if applicable)
- **TAM/SAM/SOM**: Total, serviceable, obtainable market sizing
- **Growth rate**: Historical and projected CAGR
- **Key trends**: Technology, regulatory, behavioral shifts
- **Market structure**: Fragmented vs consolidated, barriers to entry

## 3. Competitive Landscape
| Competitor | Positioning | Strengths | Weaknesses | Market Share |
|-----------|-------------|-----------|------------|-------------|
| | | | | |

- Porter's Five Forces assessment
- Competitive moats: network effects, switching costs, brand, scale, data

## 4. SWOT Analysis
| Strengths | Weaknesses |
|-----------|------------|
| | |

| Opportunities | Threats |
|--------------|---------|
| | |

## 5. Financial Analysis (if data available)
- Revenue model and unit economics
- Key metrics: CAC, LTV, LTV/CAC ratio, payback period
- Gross margin, contribution margin
- Burn rate and runway (for startups)
- Benchmarks against industry standards

## 6. Strategic Options
For each option:
- Description (what)
- Rationale (why)
- Investment required (how much)
- Expected outcome (ROI, timeline)
- Risk/downside

## 7. Recommendation
- Primary recommendation with supporting logic
- Key assumptions that must hold true
- Metrics to track (leading and lagging indicators)
- Decision timeline and next steps

## Output Formats
- **Executive brief**: 1-page summary for leadership
- **Detailed analysis**: Full structured report
- **Slide outline**: 10-slide presentation structure
