# Proposal Builder — Create a winning sales proposal

Build a structured proposal for "$ARGUMENTS":

## 1. Executive Summary (1 page max)
- Client's situation and key challenges (show you listened)
- Proposed solution in business terms (not features)
- Expected outcomes with quantified impact
- Investment overview and timeline
- Why now (urgency drivers)

## 2. Understanding Your Challenges
Map each client pain point to business impact:

| Challenge | Business Impact | Current Cost of Inaction |
|-----------|----------------|------------------------|
| | | |

This section proves you understand their world before talking about yours.

## 3. Proposed Solution
For each challenge, map to a solution component:

| Challenge | Solution Component | Outcome |
|-----------|--------------------|---------|
| | | |

- Architecture/approach overview (visual diagram if possible)
- Implementation phases with milestones
- What's included vs out of scope (be explicit)

## 4. Social Proof
- 2-3 relevant case studies (similar industry/size/challenge)
- Quantified results: "Company X achieved Y% improvement in Z months"
- Client testimonials (with permission)
- Relevant logos/client list

## 5. Investment
| Component | Description | Price |
|-----------|-------------|-------|
| | | |
| **Total** | | |

- Pricing options (if applicable): packages, tiers, or modular
- Payment terms
- ROI calculation: payback period and 3-year value
- Comparison to cost of alternatives (including doing nothing)

## 6. Implementation Plan
| Phase | Duration | Activities | Deliverables |
|-------|----------|-----------|-------------|
| 1 | | | |
| 2 | | | |
| 3 | | | |

- Key milestones and success criteria
- Resource requirements (from both sides)
- Risk mitigation approach

## 7. Why Us
- 3 differentiators (not features — advantages they can't get elsewhere)
- Team/expertise relevant to their specific needs
- Commitment and support model

## 8. Next Steps
- Clear decision timeline
- Mutual action plan with dates
- Who needs to be involved
- How to proceed (sign-off process)

## Formatting Rules
- Use their language and terminology (mirror discovery notes)
- Lead every section with THEIR perspective, not yours
- Bold key numbers and outcomes
- Keep it under 10 pages (excluding appendices)
- Include a visual on every other page
