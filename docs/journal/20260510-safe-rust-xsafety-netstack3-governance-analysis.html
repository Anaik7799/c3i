Executive Summary

Learning

Safe Rust is broader than memory safety. The Netstack3 lesson is to encode arbitrary safety properties as compile-time evidence.

Rule

All Rust generated or modified by agents must be safe-by-construction and follow SRXS-001..012.

Skill

Added Claude, Gemini, and provider-neutral skills that guide invariant discovery, parser boundaries, unsafe contracts, and verification.

Codegen

Future Rust codegen must define, enforce, and consume invariants through types, not comments or late validation.
