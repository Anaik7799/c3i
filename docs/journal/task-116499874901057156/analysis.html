3. Confirmed Gaps (DO NOT author replacements)

The Rust ecosystem has no production-grade equivalents for:

  • Optics (lens/prism/traversal) — only experimental crates; pattern-match + struct-update is acceptable.
  • Algebraic effects — no Eff-style; use tower::Layer for cross-cutting.
  • Software Transactional Memory — no stm; use rpds + atomic swap.
  • Free monads / tagless final — type system can't express; use traits + tower::Service.

Authoring our own replacements is FORBIDDEN. Workarounds above are the canonical paths.
