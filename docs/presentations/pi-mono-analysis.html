Why Symbiosis?

Pi and C3I occupy complementary niches. Neither competes — they complete each other.

Pi Excels At

  • Interactive Agent UX — rich TUI, chat, sessions
  • LLM Provider Diversity — 15 providers out of box
  • Extension System — 60+ lifecycle hooks
  • Session Management — branching, forking, export
  • Web Components — drop-in ChatPanel
  • Slack Integration — MOM bot

C3I Excels At

  • Distributed Infrastructure — Zenoh mesh
  • Safety Kernel — SIL-6, Guardian, 2oo3 voting
  • Observability — OTel over Zenoh, full tracing
  • Knowledge Management — 2,679 holon Zettelkasten
  • Rule Engine — 98 RETE-UL rules, 23 domains
  • Type Safety — Gleam exhaustive ADTs
