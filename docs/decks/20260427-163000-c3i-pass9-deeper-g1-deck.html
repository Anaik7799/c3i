Why deeper G1?

  • G1-minimal (Pass 7): 5-LOC env gate skips gemma in scheduler — saved 12.6 GB.

Pass 9 fixes this: extract sa-plan-inference daemon + UDS RPC, all clients share one gemma instance.

Queueing analysis (M/M/1 + parallel)

ρ = λ/μ; W = 1/(μ−λ); L = ρ/(1−ρ)

Steady: λ=0.3, μ=1, ρ=0.3 ✓

Burst: λ=2, μ=1, ρ=2.0 → unstable

With 4-way parallelism: μ_eff=3 → ρ_burst=0.67 ✓ stable

Backpressure: max-queue=32, RST on overflow → bounded latency tail.
